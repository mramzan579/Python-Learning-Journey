# Student Result Management System

## Description
A simple Python CLI tool that calculates total marks, average, and grades for multiple students using loops and dictionaries.

## Features
- Calculates total
- Calculates average
- Assigns grade
- Uses dictionary data structure
- Beginner friendly

## Technologies
Python 3

## How to Run
python main.py

## Sample Output
Name: Ali
Total: 245
Average: 81.6
Grade: B

## Learning Goals
- loops
- conditions
- dictionaries
- clean coding
